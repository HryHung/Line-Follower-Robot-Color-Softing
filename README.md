🚀 Line Following Robot (Color-Based Sorting)
📌 Overview

A mobile robot that follows a line, detects object color (Red/Blue), and automatically delivers it to the correct destination.

⚙️ Features
Line following using IR sensors
Color detection (Red / Blue)
PID-based motor control
Autonomous navigation & decision-making
🧠 How It Works
Follow line to loading area
Detect object placement
Identify color
Navigate to corresponding destination
Stop at endpoint
🔧 Hardware
Microcontroller (STM32 / ESP32)
IR sensor array (TCRT5000)
Color sensor
DC motors + driver (TB6612FNG)
Battery system
📊 Specs
Speed ≥ 0.1 m/s
Tracking error: ±3 mm
Stopping error: ±5 mm
Load: ~1 kg
